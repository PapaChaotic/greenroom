Settings

Wayland desktops need the GlobalShortcuts portal for these to work globally (KDE: approve the prompt, then manage in System Settings → Shortcuts). Universal fallback: bind a system shortcut that runs GreenRoom.AppImage --hud or --mic — it controls the running app from anywhere.

Sets the bitrate ceiling Xbox may use — the stream still adapts down on a weak connection. Applies to the next game you launch. Smooth 60 fps needs Maximum plus hardware decoding below.

Takes effect after restarting GreenRoom. NVIDIA cards also need the system package libva-nvidia-driver (Nobara/Fedora: sudo dnf install libva-nvidia-driver), and hardware decoding on NVIDIA relaxes Chromium's GPU-process sandbox (see SECURITY.md — renderers stay sandboxed; pick Software to keep it). Choose Software if you see graphical glitches.

Boosts the game stream louder than normal (party voices are not affected). Applies immediately. If everything is quiet, also check GreenRoom's volume in your system mixer.
